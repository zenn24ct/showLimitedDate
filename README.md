# Year Progress - カレンダー 🌈
[普通バージョン](https://zenn24ct.github.io/showLimitedDate/normal)

[ちょっとに虹色エフェクト入れたバージョン](https://zenn24ct.github.io/showLimitedDate/update01)
年間の進捗を視覚的に美しく表示するWebアプリケーション

## 機能

- 365日（閏年は366日）を虹色のグリッドで表示
- 経過した日数を7色の虹色（赤→オレンジ→黄→緑→シアン→青→紫）で自動配色
- 今日の日付を白色でハイライト表示
- 年間進捗率をリアルタイム表示
- 統計情報（総日数、経過日数、残り日数）を表示
- 年の切り替え機能（←/→ボタン）
- レスポンシブデザイン対応（PC・スマホ両対応）

## 使い方

- HTMLファイルをブラウザで開くだけで動作
- 年の切り替えは画面右上の矢印ボタンで操作
- 各日付セルにマウスホバーすると日付を表示
- 1分ごとに自動更新

## デザイン特徴

- ダークモードベースの洗練されたUI
- 虹色グラデーションアニメーション
- ガラスモーフィズム風のカードデザイン
- スムーズなホバーエフェクト
- パルスアニメーションで今日を強調

## 技術仕様

- 純粋なHTML/CSS/JavaScript（ライブラリ不要）
- CSS Grid レイアウト
- CSS アニメーション・トランジション
- 閏年自動判定
- ローカルタイムゾーン対応

## ブラウザ対応

- Chrome、Firefox、Safari、Edge の最新版
- モダンブラウザ推奨（CSS Grid、CSS Variables 対応）
